# 拾字

拍照辨識文字、建立字卡，並依第 1、3、7 天節奏複習的手機網頁 App。

## 發布到 GitHub Pages

1. 在 GitHub 建立一個新的 public repository，例如 `shizi`。
2. 將本資料夾內的所有檔案上傳到 repository 的 `main` branch，包含 `.github` 資料夾。
3. 開啟 repository 的 **Settings > Pages**。
4. 在 **Build and deployment > Source** 選擇 **GitHub Actions**。
5. 到 **Actions** 頁籤等待 `Deploy GitHub Pages` 顯示綠色勾勾。
6. 網址會是 `https://你的帳號.github.io/shizi/`。

在 iPhone 使用 Safari 開啟網址，再點「分享 > 加入主畫面」。
